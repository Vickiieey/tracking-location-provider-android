# STEPPER - Premium Footwear Website

A modern, responsive shoe brand website built with HTML, CSS, and JavaScript. This project showcases a fictional premium footwear brand called "STEPPER" with a beautiful, interactive design.

## 🌟 Features

### Design & User Experience
- **Modern & Responsive Design**: Fully responsive layout that works on all devices
- **Smooth Animations**: CSS animations and transitions for enhanced user experience
- **Interactive Elements**: Hover effects, parallax scrolling, and dynamic content
- **Accessibility**: Keyboard navigation support and focus indicators

### Sections
1. **Hero Section**: Eye-catching landing area with call-to-action buttons
2. **Products Showcase**: Display of footwear collection with pricing
3. **About Section**: Brand story and key features
4. **Contact Form**: Interactive contact form with validation
5. **Footer**: Social links and additional information

### Interactive Features
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Add to Cart**: Interactive product cards with cart functionality
- **Form Validation**: Contact form with email validation
- **Notifications**: Toast notifications for user feedback
- **Parallax Effects**: Subtle parallax scrolling in hero section

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. That's it! The website is ready to use

### File Structure
```
shoe-brand-website/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md           # Project documentation
```

## 🎨 Customization

### Colors
The website uses a modern color palette:
- Primary Blue: `#3498db`
- Secondary Orange: `#f39c12`
- Dark Blue: `#2c3e50`
- Success Green: `#27ae60`
- Error Red: `#e74c3c`

### Branding
To customize for your own brand:
1. Update the brand name "STEPPER" in `index.html`
2. Replace Font Awesome icons with your own images
3. Modify colors in `styles.css`
4. Update content and contact information

### Adding Products
To add more products:
1. Duplicate the product card structure in `index.html`
2. Update product information (name, price, description)
3. Choose appropriate Font Awesome icons or add images
