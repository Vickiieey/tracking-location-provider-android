# SoleMate Shoe Brand Landing Page

This is a simple, static landing page for the fictional shoe brand **SoleMate**.

## Running Locally

1. Navigate to the project directory:
   ```bash
   cd shoe_brand
   ```
2. Start a simple HTTP server (Python 3):
   ```bash
   python3 -m http.server 8000
   ```
3. Open your browser and go to `http://localhost:8000/index.html`.

Alternatively, you can simply open `index.html` directly in your browser, though some resources (like Google Fonts) are fetched online and work best when served via HTTP.
