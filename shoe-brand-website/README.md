# Stride - Premium Footwear Website

A modern, responsive shoe brand website built with HTML, CSS, and JavaScript. This project showcases a fictional premium footwear brand called "Stride" with a clean, professional design and interactive features.

## Features

### 🎨 Design
- **Modern & Clean**: Professional design with gradient backgrounds and smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Interactive**: Hover effects, smooth scrolling, and dynamic content
- **Accessible**: Proper semantic HTML and keyboard navigation support

### 🚀 Functionality
- **Navigation**: Fixed navigation bar with smooth scrolling to sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Product Showcase**: Interactive product cards with "Add to Cart" functionality
- **Contact Form**: Functional contact form with validation
- **Notifications**: Toast notifications for user feedback
- **Animations**: Scroll-triggered animations and parallax effects

### 📱 Responsive Sections
1. **Hero Section**: Eye-catching landing area with call-to-action buttons
2. **Products**: Grid layout showcasing different shoe categories
3. **About**: Company information with feature highlights
4. **Contact**: Contact form and business information
5. **Footer**: Social links and additional navigation

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. The website will load with all functionality intact

### Local Development
If you want to run this locally for development:

1. Navigate to the project directory:
   ```bash
   cd shoe-brand-website
   ```

2. Open the project in your preferred code editor

3. Use a local server (optional but recommended):
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

4. Open `http://localhost:8000` in your browser

## Customization

### Branding
- Update the brand name "STRIDE" in the HTML and CSS
- Modify colors in the CSS variables
- Replace placeholder content with your actual brand information

### Content
- Add real product images instead of Font Awesome icons
- Update product descriptions and pricing
- Modify contact information and social media links

### Styling
- Adjust the color scheme in `styles.css`
- Modify typography by changing the Google Fonts import
- Customize animations and transitions

## Performance Features

- **Optimized Images**: Uses Font Awesome icons instead of heavy images
- **Minimal Dependencies**: Only external dependencies are Google Fonts and Font Awesome
- **Smooth Animations**: Hardware-accelerated CSS transitions
- **Lazy Loading**: Intersection Observer for scroll-triggered animations
